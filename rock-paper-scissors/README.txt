## Proejct name 
Rock-paper-scissors

## Description
This is an application that allows you to play rock-paper-scissors in the console of an html written in html and javascript

## Usage
when the user starts the game, he needs to click a button of either "Rock", "Paper" or "Scissors", the computer will randomly generated once of the option and compare
who is the winner, then display the winner in a message
